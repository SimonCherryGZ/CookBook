
CookBook
===================================
基于[MobApi菜谱接口](http://api.mob.com/#/apiwiki/cookmenu)的Android应用
  
实现了以下功能：
1. 按菜谱分类查询。
2. 按关键词查询。
3. 查询历史。 
4. 菜谱收藏。
5. 最近浏览菜谱记录。
